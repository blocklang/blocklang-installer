# installer

从软件中心下载软件，并在应用服务器上安装软件环境和软件。
本软件部署在应用服务器上。

## 功能

![结构图](images/installer.png)

1. 从软件中心下载JDK和Jar文件；
1. 在应用服务器上安装JDK和Jar文件；
1. 启动Jar文件

### 功能点说明
搭建一个 CLI 项目。
在控制台中输入 `installer update` 命令后，在控制台打印出“更新成功”。

```
$ installer update
更新成功
```
