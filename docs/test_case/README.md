# 测试用例

## 数据有效性校验

1. 输入无效的 URL，是否会提示无法连接
2. 输入无效的 token，是否会准确提示根据 token 无法获取项目
3. 输入无效的端口号，是否会提示不是有效的数字
4. 输入已被占用的端口号，是否会提示端口号被占用
